Url Base: http://norson17:8081/

Endpoint Metodo Parametros Respuesta
/api/auth/register POST { "email","password","password_confirmation","empleado","rfc","curp" } Regresa un mensaje y un objeto con el usuario registrado
/api/auth/login POST { "email","password", "idempleado" } Regresa un token y un objeto con el usuario registrado
/api/auth/user-profile GET Token en el Header Autorization: Regresa el objeto usuario.
/api/auth/refresh POST Token en el Header Autorization: Destruye el token actual y regresa uno nuevo.
/api/auth/logout POST Token en el Header Autorization: Cierra la sesión del usuario destruyendo el token.
/api/auth/reset-password-request POST { "email","url" } Envia un correo al usuario con un token valido para proceder al cambio de contraseña
/api/auth/change-password POST { "token", "email", "password", "password_confirmation" } Actualiza la contraseña del usuario
